# Carrito-De-compras
TN STORE — Trabajo Práctico de Programación Descripción:

Este proyecto es una tienda web simple llamada TN STORE, desarrollada como trabajo práctico de programación. El objetivo principal es mostrar la funcionalidad del carrito de compras, utilizando HTML, CSS y JavaScript.

Funcionalidades principales:

Visualización de productos: se muestran varios productos con su nombre, precio y una insignia si son “últimas unidades”.

Agregar al carrito: cada producto tiene un botón que permite sumarlo al carrito.

Modificar cantidades: dentro del carrito se pueden aumentar, disminuir o eliminar productos.

Vaciar carrito: botón para limpiar todo el carrito.

Calcular total: muestra automáticamente el total actualizado según los productos agregados.

Carrito persistente: los datos se guardan en el localStorage, por lo que al recargar la página el carrito se mantiene.

Buscador de productos: permite filtrar los productos por nombre.

Simulación de pago: al presionar “Pagar” se muestra un mensaje con el total de la compra.

Tecnologías utilizadas:

HTML5 — estructura de la página

CSS3 (Bootstrap 5 + estilos propios) — diseño responsive y moderno

JavaScript (ES6) — lógica del carrito, buscador y almacenamiento local

Archivos principales

index.html → estructura principal de la tienda

style.css → estilos personalizados

app.js → lógica del carrito y funciones interactivas
